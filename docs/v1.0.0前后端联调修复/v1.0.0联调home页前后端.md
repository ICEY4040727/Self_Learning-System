### 3.3 补充 SageInfo 的获取逻辑

在 `archive.py` 的 `get_worlds` 和 `get_world` 端点中，补充 sages 数组的查询逻辑：

```python
def _get_world_sages(db: Session, world_id: int) -> list[SageInfo]:
    links = db.query(WorldCharacter).filter(
        WorldCharacter.world_id == world_id,
        WorldCharacter.role == "sage"
    ).all()
    
    result = []
    for link in links:
        char = db.query(Character).filter(Character.id == link.character_id).first()
        if char:
            result.append(SageInfo(
                id=char.id,
                name=char.name,
                title=char.personality or "",
                symbol=char.avatar or "☉",
                color=char.sprites.get("color", "#ffd700") if char.sprites else "#ffd700",
                accentColor=char.sprites.get("accentColor", "#fbbf24") if char.sprites else "#fbbf24"
            ))
    return result
```

## 十三、技术备注

### 13.1 数据库关系

```
User (1) ── (N) World
World (1) ── (N) WorldCharacter ── (1) Character
World (1) ── (N) Course
```

### 13.2 认证方式

所有接口（除 `/api/auth/*` 外）均需要 Bearer Token 认证：

```
Authorization: Bearer {token}
```

### 13.3 错误处理

统一使用 HTTPException 返回错误：
- 400: 请求参数错误
- 401: 未认证
- 403: 无权限
- 404: 资源不存在
- 500: 服务器错误
