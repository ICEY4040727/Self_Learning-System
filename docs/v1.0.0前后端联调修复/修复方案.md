## 二、修复详情

### P0-1: 数据库迁移 - Character 表添加经验值和等级字段

**问题描述**：
`Character` 模型缺少 `experience_points` 和 `level` 字段，导致 `POST /api/character/{id}/levelup` 接口无法正常工作。

**修复方案**：
在 `backend/alembic/versions/` 目录下创建新的迁移文件。

**迁移内容**：
```python
# 新增字段
experience_points = Column(Integer, default=0)  # 当前经验值
level = Column(Integer, default=1)  # 当前等级
```

**验收标准**：
- [ ] 迁移文件创建成功
- [ ] 迁移可正常执行
- [ ] `CharacterLevelupResponse` 包含正确的字段

### P0-2: Provider 名称不一致修复

**问题描述**：
- 前端 `Settings.vue` 定义：`openai`, `claude`, `anthropic`, `google`, `deepseek`
- 后端支持：`claude`, `openai`, `local` (含 ollama)

**修复方案**：
统一前端 provider 列表，移除后端不支持的 `anthropic` 和 `google`，添加 `local`（对应 ollama）。

**修改文件**：
`frontend/src/views/Settings.vue:209-215`

**修改前**：
```typescript
const providers: Array<{ value: LLMProvider; label: string }> = [
  { value: 'openai', label: 'OpenAI' },
  { value: 'claude', label: 'Claude' },
  { value: 'anthropic', label: 'Anthropic' },
  { value: 'google', label: 'Google' },
  { value: 'deepseek', label: 'DeepSeek' },
]
```

**修改后**：
```typescript
const providers: Array<{ value: LLMProvider; label: string }> = [
  { value: 'claude', label: 'Claude' },
  { value: 'openai', label: 'OpenAI' },
  { value: 'deepseek', label: 'DeepSeek' },
  { value: 'local', label: '本地模型 (Ollama)' },
]
```

**验收标准**：
- [ ] Settings.vue 的 provider 列表与后端一致
- [ ] `LLMProvider` 类型定义同步更新

### P1-1: 扩展 CourseResponse 模型

**问题描述**：
`CourseResponse` 缺少 `progress` 和 `icon` 字段，前端世界选择页无法显示课程进度。

**修复方案**：
扩展 `backend/api/routes/archive.py` 中的 `CourseResponse` 模型。

**修改位置**：
`backend/api/routes/archive.py:160-164`

**修改前**：
```python
class CourseResponse(CourseCreate):
    id: int

    class Config:
        from_attributes = True
```

**修改后**：
```python
class CourseResponse(CourseCreate):
    id: int
    progress: float | None = None  # 0.0 - 1.0 课程完成度
    icon: str | None = None  # 课程图标

    class Config:
        from_attributes = True
```

**数据填充逻辑**（在 `_build_world_response` 中）：
```python
# 获取课程进度
course_progress = db.query(ProgressTracking).filter(
    ProgressTracking.course_id == course.id,
    ProgressTracking.user_id == current_user.id
).first()
progress = course_progress.mastery_level / 100.0 if course_progress else None

# 获取课程图标（从 world 的 scenes 或默认）
icon = "📖"  # 默认图标

course_list.append(CourseResponse(
    id=course.id,
    world_id=course.world_id,
    name=course.name,
    description=course.description,
    target_level=course.target_level,
    progress=progress,
    icon=icon,
))
```

**验收标准**：
- [ ] `CourseResponse` 包含 `progress` 和 `icon` 字段
- [ ] 前端 Worlds.vue 可正常接收并显示进度
