## 三、核心设计点

### 3.1 Header 区域

| 元素 | 说明 |
|------|------|
| 返回按钮 | 左侧，指向 `/home/worlds` |
| 世界名称 | 居中显示 |
| 编辑按钮 | 右侧，弹出 EditWorldModal |
| 删除按钮 | 右侧，弹出确认对话框 |

**样式**：毛玻璃背景 `backdrop-filter: blur(10px)`

### 3.2 世界信息卡片

- 左侧：世界图标/符号（64x64 圆形）
- 右侧：世界名称 + 描述 + 分隔线
- 背景：从 `world.scenes?.background` 获取，无则用默认图

### 3.3 旅者 Section

旅者（Traveler）属于特定世界，通过 `WorldCharacter` 关联表绑定。

| 功能 | 说明 |
|------|------|
| 显示 | 当前旅者卡片（头像、名称、称号） |
| 切换 | 下拉菜单切换旅者身份 |
| 编辑 | 点击编辑旅者信息 |
| 创建 | 添加新旅者到当前世界 |

### 3.4 知者 Section

| 功能 | 说明 |
|------|------|
| 显示 | 使用 CharacterCard 组件 |
| 编辑 | 点击编辑图标 |
| 删除 | 点击删除图标 |
| 添加 | + 按钮弹出创建弹窗 |

### 3.5 课程 Section

| 功能 | 说明 |
|------|------|
| 显示 | 课程卡片列表 + 进度条 |
| 点击 | 进入 Learning 页面 |
| 添加 | + 按钮弹出创建弹窗 |

**注意**：回忆选择（Checkpoint）不在此页面，移至课程详情页。

## 四、后端接口

### 4.1 获取世界详情
```
GET /api/worlds/:worldId
```

响应：
```json
{
  "id": 1,
  "name": "哲学世界",
  "description": "探索西方哲学的奥秘",
  "symbol": "🏛️",
  "scenes": { "background": "/assets/world-bg-1.jpg" },
  "travelers": [
    { "id": 101, "name": "求知者", "title": "新手", "avatar": "...", "color": "..." }
  ],
  "sages": [
    { "id": 1, "name": "苏格拉底", "title": "哲学之父", "symbol": "?", "color": "..." }
  ]
}
```

### 4.2 编辑世界
```
PUT /api/worlds/:worldId
```

### 4.3 删除世界
```
DELETE /api/worlds/:worldId
```

### 4.4 获取课程列表
```
GET /api/worlds/:worldId/courses
```
